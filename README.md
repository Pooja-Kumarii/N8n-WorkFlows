# 🔁 n8n Automation Workflows

A growing collection of practical, real-world automation workflows built with **n8n** — covering AI agents, content generation, lead generation, news automation, and more.

Each workflow is saved as an importable `.json` file. Just download and drop it into your n8n editor and it is ready to run.

---

## 🚀 What is n8n?

[n8n](https://n8n.io/) is a powerful open-source workflow automation tool that lets you connect apps, APIs, and services without writing much code. Think of it like building automations visually — with full flexibility to go deep with code when needed.

## 📁 Workflows Included

| File | Type | Description |

| `Final Rag Agent.json` | AI Agent | A complete RAG (Retrieval-Augmented Generation) agent that fetches relevant context before answering queries |

## 🔧 How to Use Any Workflow

1. Open your [n8n editor](https://n8n.io/) (cloud or self-hosted)
2. Click **"+"** → **"Import from File"**
3. Select the `.json` workflow file from this repo
4. Add your own API keys / credentials where required
5. Click **Execute** and watch it run!
